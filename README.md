# 🎬 Movie Recommendation System

A Machine Learning based movie recommendation system built using Python and Streamlit.

## 🚀 Live Demo

[Click Here](https://movie-recommender-system-nwghkiukjrqchr5fikyzav.streamlit.app/)

## 📌 Features

- Movie recommendations
- Content-based filtering
- Cosine similarity
- Streamlit web interface
- Poster integration
- Cloud deployment

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Requests API

## 🧠 Machine Learning Concept

This project uses Content-Based Filtering.

Movies are converted into vectors using:
- genres
- keywords
- cast
- overview

Cosine similarity is then used to recommend similar movies.

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deployment

Deployed using Streamlit Community Cloud.
